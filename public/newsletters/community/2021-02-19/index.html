Inter-Blockchain Communication
IBC protocol: the standard in blockchain interoperability

The IBC protocol provides the foundation for a new token economy, permitting anyone around the world to accomplish quick and secure cross-chain exchanges. It is the industry's first framework for a standardization of blockchain development from protocol level to applications.

IBC can be used for a wide range of cross-chain applications, which include token transfers, atomic swaps, multi-chain smart contracts and data and code sharding of various kinds. The newly launched IBC protocol website explains how it works and what it means for the blockchain industry.
