The CosmWasm Cosmos Hub proposal passed


The Cosmos Hub‚Äôs 25th governance proposal to build a gateway with CosmWasm passed with flying colors. üéä It had one of the highest delegator turnouts with 5% of the whole network.


"[We] think this feature may help the Cosmos Hub establish itself as the center of the IBC ecosystem."

‚Äì Validator Chorus One


What will the integration entail?


  • Governance control to all aspects of the CosmWasm contract lifecycle with Cosmos Hub compatibility for governance to control code upload, contract instantiation, upgrades, and destruction.

  • Contract upgrades along with migrations (also allowing orderly shutdowns) via a governance vote.

  • Launching a pre-release testnet environment for feedback and sample contracts to demo on the testnet, along with some migration scenarios.
